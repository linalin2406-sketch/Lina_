# 📚 Книжный портал — новости и тренды книжного рынка

Сайт, который сам собирает книжные новости из открытых RSS-лент по всему интернету
и делит их на две вкладки: **🇷🇺 Россия** и **🌍 За рубежом**. Есть поиск, фильтр по
источникам и блок трендов (самые частые темы в свежих заголовках).

Всё работает локально на вашем компьютере — без регистраций, ключей и оплаты.

---

## 🚀 Как запустить (пошагово)

> Нужно сделать это один раз при установке. Все команды вводятся в приложении
> **«Терминал»** (оно уже есть на macOS: Finder → Программы → Утилиты → Терминал).

### Шаг 1. Открыть папку проекта
Скопируйте строку ниже в Терминал и нажмите Enter:
```
cd /Users/lina/book-portal
```

### Шаг 2. Установить нужные библиотеки (один раз)
```
pip3 install -r requirements.txt
```
Дождитесь, пока установка завершится (появится обычная строка ввода).

### Шаг 3. Запустить портал
```
python3 app.py
```
Вы увидите сообщение:
```
Запускаю портал. Откройте в браузере:  http://127.0.0.1:8000
```

### Шаг 4. Открыть сайт
Откройте браузер (Safari, Chrome…) и перейдите по адресу:

**http://127.0.0.1:8000**

Готово! При первом запуске новости подгрузятся за несколько секунд — если список
пустой, подождите чуть-чуть и нажмите на сайте кнопку **«Обновить сейчас»**.

### Как остановить
Вернитесь в Терминал и нажмите **Ctrl + C**.
Чтобы запустить снова — повторите Шаг 1 и Шаг 3.

## 🧭 Что умеет сайт

- **Вкладки Россия / За рубежом / Все** — переключение одним кликом.
- **Поиск** — введите слово, и останутся только подходящие новости.
- **Фильтр по источнику** — выпадающий список справа от поиска.
- **Тренды** — облако тем справа; нажатие на тему фильтрует новости по ней.
- **Обновить сейчас** — кнопка вверху, скачивает свежие новости немедленно.
- **Автообновление** — пока сайт запущен, новости сами обновляются каждый час.

## 🔧 Как добавить или убрать источник новостей

Откройте файл **`feeds.py`** в любом текстовом редакторе. Внутри — два списка:
`RU_FEEDS` (Россия) и `INT_FEEDS` (за рубежом). Каждая строка выглядит так:

```python
("Название источника", "адрес RSS-ленты", False),
```

- **Добавить** источник — впишите новую такую строку.
- **Убрать** источник — удалите строку или поставьте `#` в её начале.
- Последнее значение:
  - `False` — брать из ленты **все** новости (лента и так про книги);
  - `True`  — брать **только про книги** (для больших новостных лент вроде ТАСС;
    отбор идёт по ключевым словам из списка `BOOK_KEYWORDS` в том же файле).

После изменений остановите портал (Ctrl + C) и запустите заново (Шаг 3).

## ❓ Частые вопросы

**Какая-то лента «не ответила» — это страшно?**
Нет. Сайты иногда меняют адреса RSS или временно недоступны. Приложение просто
пропускает такие ленты и показывает остальные. Если источник перестал работать
навсегда — замените его адрес в `feeds.py`.

**Где хранятся новости?**
В файле `books.db` рядом с приложением (это база данных SQLite). Его можно удалить —
он создастся заново при следующем запуске.

**Можно ли открыть сайт с телефона или другого компьютера?**
Да. Чтобы сайт стал доступен по ссылке из интернета, следуйте пошаговой
инструкции в файле **`DEPLOY.md`** (бесплатно, через GitHub + Render.com).

## 🗂 Из чего состоит проект

| Файл | Что делает |
|------|------------|
| `app.py` | Запускает сайт и фоновое обновление новостей |
| `feeds.py` | Список RSS-лент (здесь вы добавляете/убираете источники) |
| `fetcher.py` | Скачивает и разбирает ленты, считает тренды |
| `database.py` | Хранит новости в локальной базе `books.db` |
| `static/` | Сам сайт: страница, стили, скрипты |
| `requirements.txt` | Список библиотек для установки |
